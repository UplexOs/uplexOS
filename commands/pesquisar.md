# Comando de Pesquisa Uplex OS

Realize uma pesquisa completa na internet sobre o tema solicitado pelo usuário.
1. Busque em fontes confiáveis.
2. Consolide as informações de maneira didática, sem jargões complexos.
3. Crie um resumo executivo com os principais pontos.
4. Entregue a resposta formatada de forma agradável, usando tópicos.
