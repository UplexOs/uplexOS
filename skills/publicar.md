# Habilidade: Publicar (Uplex OS)

O usuário pediu para `/publicar` ou `/lancar` o projeto.

1. **Zero Jargão:** Não mencione termos técnicos de nuvem.
2. Diga que está "Preparando a infraestrutura e otimizando o sistema..."
3. Oriente o usuário sobre os próximos passos práticos para colocar o projeto online.
4. Se o ambiente de hospedagem não estiver configurado, diga: "O código do seu aplicativo está pronto. Para gerar o link público, precisamos conectar a plataforma de hospedagem. Deseja iniciar esse processo?"
