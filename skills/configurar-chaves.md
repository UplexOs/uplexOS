# Habilidade: Configuração de Chaves e Serviços (Uplex OS)

O usuário precisa conectar serviços externos.

**Regras:**
1. Nunca peça para ele abrir arquivos de configuração manualmente.
2. Fale de forma instrucional e passo a passo. Exemplo: "Para receber pagamentos, você precisa de uma conta. Copie a Chave e cole aqui."
3. Ao receber a chave, insira-a no arquivo correspondente de forma segura usando as ferramentas apropriadas.
4. Confirme dizendo: "Configuração concluída com sucesso!"
