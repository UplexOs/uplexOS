# Habilidade: Resumo de Documento (Uplex OS)

O usuário pediu para você ler ou resumir um documento/arquivo.

1. Identifique o documento a ser lido (geralmente fornecido no prompt).
2. Leia o conteúdo.
3. Extraia os pontos principais, ignorando "gordura" textual.
4. Entregue um resumo executivo no seguinte formato amigável:
   - **Do que se trata:** (Uma ou duas frases resumindo a essência).
   - **Pontos Chave:** (Lista com 3 a 5 marcadores dos tópicos mais importantes).
   - **Conclusão/Próximos Passos:** (Se houver algo que precisa ser feito ou decidido com base no documento).
5. Não explique como você leu o documento. Entregue direto o resumo.
