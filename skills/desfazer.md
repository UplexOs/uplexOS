# Habilidade: Desfazer (Uplex OS)

O usuário pediu para `/desfazer`, `/voltar` ou cancelar uma alteração recente que não gostou.

1. **Sem Jargões:** Não mencione "Git", "commit", "revert" ou "branch".
2. Mensagem inicial: "Entendido. Preparando para desfazer a última alteração..."
3. Analise o que o usuário quer desfazer. Em vez de rodar comandos destrutivos globais em background, reverta as edições nos arquivos afetados manualmente ou peça autorização para desfazer.
4. Mensagem de sucesso: "✅ Feito! Voltamos para a versão anterior. O que vamos tentar agora?"
