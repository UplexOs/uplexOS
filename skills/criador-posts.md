# Habilidade: Criador de Conteúdo para Redes Sociais (Uplex OS)

O usuário deseja criar conteúdo para postar em uma rede social.

1. Identifique a rede social desejada (se não for informada, crie para Instagram como padrão).
2. Entenda o tema e o tom desejado pelo usuário.
3. Crie um post focado, atrativo e bem espaçado.
4. Inclua sugestões de hashtags relevantes no final.
5. Se for para Instagram, sugira brevemente o que poderia ser a imagem ou vídeo do post.
6. Entregue apenas o texto do post, pronto para copiar.
