# UplexOS Knowledge Vault

Abra [[UplexOS]] como núcleo, [[Timeline Map]] para navegação temporal ou `Neural Timeline.canvas` para a visão horizontal fixa.

## Estrutura neural

- `00-home`: núcleo e mapas principais.
- `01-timeline`: eras de maturidade operacional.
- `20-governance` a `50-operations`: clusters de domínio.
- `80-agents`: catálogo sincronizado das 48 skills.
- `85-mcp`: capacidades executáveis e adaptadores planejados.
- `90-projects`: memória isolada por projeto.
- `95-templates`: decisões, runbooks e findings.

## Configuração visual

O Graph View contém grupos de cor por domínio. Para ativar o brilho neural, abra **Settings → Appearance → CSS snippets** e habilite `neural-timeline`.

## Fontes de verdade

Conhecimento e decisões ficam no vault. Estado do runtime, aprovações e evidências permanecem estruturados. Secrets nunca pertencem aqui. Consulte [[Knowledge Map]], [[Data Classification]] e [[Evidence Model]].
