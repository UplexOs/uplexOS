# Database Intelligence

Planned read-only capabilities: detect engine and version, inspect schema and indexes, explain queries, review constraints and RLS, analyze migrations and estimate lock risk.

Write capabilities—migration apply and rollback—must remain separate and approved. No database adapter is active today.
