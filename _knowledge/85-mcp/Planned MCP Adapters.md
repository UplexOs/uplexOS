# Planned MCP Adapters

Disabled in `.mcp/registry.json` until real configuration exists:

- Git: branch policy, secret scan and approved push.
- Browser: Playwright environment and authorized targets.
- Database: connection, read-only defaults and migration approvals.
- Secrets: approved vault provider without returning values.
- Deploy: provider credentials, gates, health and rollback.
- Analytics: metric catalog, lineage and source access.
- Marketing: provider adapters and publication approvals.

A planned node is not an available capability. Related: [[MCP Map]], [[Era 4 - Evolution]].
