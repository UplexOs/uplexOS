# Security MCP

Detects installed security capabilities and runs bounded local heuristic triage.

Tools: `security.posture`, `security.heuristic_review`.

The heuristic review is not SAST, SCA or DAST. Professional scans require installed and configured tools. Related: [[Security Toolchain]], [[Security Engineer]], [[Finding Management]].
