# Files MCP

Provides sandboxed project listing, reading and atomic writing with SHA-256 conflict checks.

Tools: `files.list`, `files.read`, `files.write`.

No arbitrary host filesystem access. Related: [[MCP Map]], [[Implementation Practice]], [[Evidence Model]].
