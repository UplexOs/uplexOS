# Testing MCP

## Tools

- `testing.discover`: inspect configured test capability.
- `testing.plan`: resolve an allowlisted profile.
- `testing.run_script`: execute one configured profile check.
- `testing.smoke_http`: verify an expected localhost response.

The server does not accept arbitrary shell commands or install dependencies. External targets require another authorized adapter.
