# Knowledge MCP

Searches, reads and creates provenance-aware Markdown notes in the Obsidian vault.

Tools: `knowledge.search`, `knowledge.read_note`, `knowledge.create_note`.

Current search is textual, not semantic. It does not overwrite an existing note ID. Related: [[Knowledge Map]], [[Data Classification]], [[Memory Keeper]].
