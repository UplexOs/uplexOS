# Projects Map

Each project receives an isolated cluster under `90-projects/<project-id>/` or through generated projections from `_projetos/<project-id>/`.

Recommended nodes: project index, confirmed facts, decisions, architecture, risks, handoffs, releases and incidents.

Project state remains in the runtime, not in manually edited notes. Use [[Project MCP]], [[Decision Management]] and [[Evidence MCP]].
