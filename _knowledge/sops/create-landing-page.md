# SOP: Criação de Landing Page de Alta Conversão

**Procedimento Operacional Padrão** obrigatório para execução de qualquer projeto focado em captura e vendas.

## Procedimento de Execução
1. **Descoberta:** Definir a Persona e o Objetivo (Vender? Capturar Lead?).
2. **Copywriting:** Elaborar o arquivo longo de copy (8 dobras). Proibido textos curtos.
3. **UI System:** Gerar o DNA Visual. Definir Cores, Fontes (Geist/Inter) e Atmosfera (Glassmorphism, Dark).
4. **Interface (Frontend):** 
   - Renderizar o Hero com Spline/3D ou Foto Absurda do Unsplash.
   - Criar Bento Grid.
   - Criar Accordions de FAQ.
5. **SEO Configuration:** Injetar Metadata API, OpenGraph e JSON-LD.
6. **Revisão:** Rodar Auto-Auditoria. A média do site deve ser 9.5+.
7. **Deploy:** Publicar na nuvem.
