# Runtime Smoke Testing

Verify an expected local HTTP status and optional body marker after the user starts the application through its normal documented process.

The built-in tool permits only localhost targets. Preview, staging and production targets require a separate authorized browser/network adapter.

Tool: `testing.smoke_http` in [[Testing MCP]].
