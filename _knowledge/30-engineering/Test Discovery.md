# Test Discovery

Detect package manager, scripts, test files and configuration for Jest, Vitest, Playwright or Cypress. Discovery executes nothing and does not infer coverage from file presence.

Tool: `testing.discover` in [[Testing MCP]]. Next: [[Test Profiles]].
